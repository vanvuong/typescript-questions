# Install prerequisites

* Install [mvn](https://github.com/creationix/nvm)
* Install node 10: `nvm install 10`
* Use node 10: `nvm use 10`
* Install yarn: `npm i -g yarn`
* Clone this repo

# For each assignment

* Install dependencies: `yarn install`
* Read the README.md file of the assignment
* Implement a solution so that all test pass.
* Run the tests: `yarn test`
